Shortcuts

Ctrl+Enter : Execute selected block or single line of code, where the cursor is.
Shift+Ctrl+Enter : Execute selected block or single line of code at the start of the next measure
Ctrl+.: Clear all existing audio, visuals and interfaces.

Please run this website in the modern web-browser. Accroding to the Gibber website, "Gibber runs best in Chrome, followed by Firefox and Safari. Some parts run in Opera; IE should be supported in the future when they complete support for the Web Audio. To run the graphics examples you need a computer that is WebGL capable; this page will tell you whether or not you can run WebGL."
